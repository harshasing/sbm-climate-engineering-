# Manokamana (Chill Wave)

TanStack Start site for **S.B.M. Climate Engineering P. Ltd.** with product catalog, blog, about page, and quote flow.

## Tech Stack

- TanStack Start + TanStack Router (file-based routing)
- React 19
- Tailwind CSS v4
- Cloudflare Vite plugin / Wrangler deploy target

## Run Locally

```bash
bun install
bun run dev
```

App runs on `http://localhost:3000`.

## Scripts

```bash
bun run dev
bun run build
bun run preview
bun run lint
bun run format
bun run check
bun run test
```

## Project Structure

- `src/routes` - Route files
- `src/components` - Reusable UI components
- `src/data/products.ts` - Product/category/subcategory dataset + flattened products
- `src/data/blogs.ts` - Blog dataset
- `public/products` - Product images (WebP)
- `public/blogs` - Blog images (WebP)
- `public/trusted-partners` - Partner logos used on About page (WebP)

## Routing Notes

- Product details use slug route: `/products/$slug`
- Blog details use slug route: `/blog/$slug` (migrated from id-based route)

## Asset Conventions

- All local site images in `public` are standardized to `.webp`
- Product image references are driven by `images` arrays on subcategories in `src/data/products.ts`
- Blog cards/details use local images from `src/data/blogs.ts`
- About page partner logos use local files from `public/trusted-partners`

## Favicon

- Favicon is logo-based:
  - `public/favicon.ico` generated from `public/logo.webp`
  - `public/favicon.png` available as PNG variant

## Deployment

Configured for Cloudflare Workers via Wrangler:

```bash
bun run deploy
```

If needed:

```bash
wrangler login
```

## Notes

- `src/routeTree.gen.ts` is generated by TanStack Router.
- If you change route files, regenerate/restart dev server as needed.
